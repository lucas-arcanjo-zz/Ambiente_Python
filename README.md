Operadores Aritméticos

   5 + 2 == 7 <-- adição              5 ** 2 == 25 <-- potencialização
   5 - 2 == 3 <-- subtração             5 // 2 == 4 <-- divisão inteira
   5 * 2 == 10 <-- multiplicação            5 % 2 == 1 <-- resto
   5 / 2 == 2,5 <-- divisão

Ordem de precêdencia 

1º ()
2° **
3º *, /, //, %
4º +, -

Exemplos

    5 + 3 * 2 == 11
    3 * 5 + 4 ** 2 == 31
    3 * (5 + 4) ** 2 == 243
